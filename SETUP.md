### 🚀 Getting Started

#### 1. Install Dependencies
```bash
npm install
# or
yarn install
# or
pnpm install
```

#### 2. Run Development Server
```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

#### 3. Open in Browser
Navigate to [http://localhost:3000](http://localhost:3000)
